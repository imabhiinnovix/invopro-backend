# Searchify

Searchify is a simple search engine built with Node.js, Express, and MongoDB. It allows users to upload files

## Features

- **File Upload:** Users can upload files to the server.
- **MongoDB Integration:** Files are stored in a MongoDB database.

## Getting Started

1. **Install dependencies:**

   ```bash
   cd searchify
   yarn
   ```

2. **Set environment variables:**

   ```bash
   MONGO_URI=mongodb://localhost:27017/searchify
   PORT=3000
   ```

3. **Start the server:**

   ```bash
   yarn start
   ```
